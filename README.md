# 🔢 React Counter App

A simple and responsive Counter application built using **React** and **Tailwind CSS**.

## 🚀 Features

- ➕ Increment counter (Maximum limit: 10)
- ➖ Decrement counter (Minimum limit: 0)
- 🔄 Reset button
- 🎨 Styled using Tailwind CSS
- 📱 Responsive design

---

## 🛠️ Built With

- React (useState Hook)
- Tailwind CSS
- JavaScript (ES6)

1. Clone the repository

---

## 🎯 How It Works

- The counter state is managed using React's `useState` hook.
- The increment function increases the count until it reaches 10.
- The decrement function decreases the count until it reaches 0.
- The reset button sets the count back to 0.
